# Portfolio content maintenance

The production site is a single static page in `index.html`. Search for the labelled comments in that file when updating content:

- `PROJECT STATUSES` — project labels, current-state wording, and status classes.
- `CAREER DATES` — experience dates and descriptions.
- `CREDENTIALS` — credential names, issuers, and completion years.
- `CONTACT LINKS` — email, telephone, LinkedIn URL, and résumé filename.

Also keep the visible contact details and the `Person` JSON-LD in the document head aligned. The canonical public address is `https://ryanburrell.github.io/`.

The site references `Ryan_Burrell_Resume_2026.pdf`, exported from the owner-approved latest résumé supplied in July 2026. When the résumé changes, replace that PDF without changing its stable public filename and re-test every résumé link. Do not add a GitHub profile or credential verification link without an owner-approved public URL.
